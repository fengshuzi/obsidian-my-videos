# 我的影视 - 使用说明

## 快速开始

### 第一步：记录观看

在你的日记文件中（如 `journals/2024-01-11.md`），使用标签记录观看：

```markdown
# 2024年1月11日

今天看了几部好片：
- #video/movie 《肖申克的救赎》 经典之作，关于希望和自由
- #video/tv 《权力的游戏》第一季 史诗级剧集
- #video/variety 《向往的生活》 放松心情的好节目
```

### 第二步：打开插件

1. 点击左侧边栏的 🎬 图标
2. 或使用命令面板搜索"打开我的影视"

### 第三步：查看记录

- 选择年份和月份查看对应时间的观看记录
- 查看统计数据了解观看习惯
- 点击日期可以跳转到对应的日记

## 标签格式详解

### 基本格式

```
#video/类型 标题 评论
```

### 推荐格式（使用书名号）

```
#video/类型 《标题》 评论内容
```

使用书名号可以更清晰地区分标题和评论。

### 示例

```markdown
# 电影
- #video/movie 《盗梦空间》 诺兰的神作，多层梦境设计精妙
- #video/movie 《星际穿越》 科幻与情感的完美结合

# 电视剧
- #video/tv 《绝命毒师》 剧情紧凑，演技炸裂
- #video/tv 《黑镜》第一季 对科技的深刻反思

# 综艺
- #video/variety 《奇葩说》 思辨与娱乐并存
- #video/variety 《向往的生活》 治愈系慢综艺
```

## 功能说明

### 1. 年月筛选

- **年份选择**：查看某一年的观看记录
- **月份选择**：
  - 选择"全年"查看整年数据
  - 选择具体月份查看该月数据

### 2. 统计卡片

- **总观看数**：显示选定时间范围内的总观看数量
- **电影**：电影观看数量
- **电视剧**：电视剧观看数量
- **综艺**：综艺观看数量

### 3. 观看记录列表

按日期倒序显示：
- 日期和星期
- 影视类型标签（不同颜色）
- 标题和评论
- 点击日期可跳转到日记

### 4. 配置管理

#### 基础设置
- **应用名称**：自定义插件名称（如"观影记录"、"追剧日志"等）
- **视频标签**：自定义标签前缀（如 `#watch`、`#看片` 等）

#### 类型管理
- 添加新类型（如"动漫"、"纪录片"等）
- 修改现有类型
- 删除不需要的类型

## 自定义配置示例

### 示例1：改用中文标签

配置：
- 视频标签：`#看片`
- 类型：`电影`、`剧集`、`综艺`

使用：
```markdown
- #看片/电影 《肖申克的救赎》 经典
- #看片/剧集 《权力的游戏》 史诗
```

### 示例2：添加更多类型

添加类型：
- `anime` - 动漫
- `documentary` - 纪录片
- `short` - 短片

使用：
```markdown
- #video/anime 《你的名字》 新海诚的代表作
- #video/documentary 《地球脉动》 震撼的自然纪录片
- #video/short 《调音师》 反转精彩
```

### 示例3：简化标签

配置：
- 视频标签：`#v`

使用：
```markdown
- #v/movie 《盗梦空间》 烧脑
- #v/tv 《绝命毒师》 精彩
```

## 常见问题

### Q: 为什么看不到记录？

A: 检查以下几点：
1. 日记文件是否在 `journals` 文件夹中
2. 文件名是否是 `yyyy-mm-dd.md` 格式
3. 标签格式是否正确（默认 `#video/类型`）
4. 年月选择是否正确

### Q: 如何修改标签格式？

A: 
1. 点击"配置"按钮
2. 在"基础设置"中修改"视频标签"
3. 保存配置
4. 在日记中使用新的标签格式

### Q: 可以导出数据吗？

A: 目前不支持导出，但所有数据都存储在你的日记文件中，可以直接查看和备份。

### Q: 支持移动端吗？

A: 支持，插件在移动端也可以正常使用。

## 性能优化

- 插件使用智能缓存机制，30秒内重复查询会使用缓存
- 文件修改会自动清除缓存
- 批量处理文件，提升加载速度

## 数据安全

- 所有数据存储在你的日记文件中
- 插件只读取和解析，不会修改原始内容
- 配置文件存储在插件目录中
