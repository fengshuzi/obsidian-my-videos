# 快速开始

## 安装插件

### 方法1：手动安装

1. 将 `obsidian-my-videos` 文件夹复制到你的 vault 的 `.obsidian/plugins/` 目录下
2. 在 Obsidian 设置中启用"我的影视"插件
3. 重启 Obsidian

### 方法2：开发模式

```bash
cd obsidian-my-videos
npm install
npm run build
npm run deploy  # 需要先配置 deploy.sh 中的路径
```

## 第一次使用

### 1. 打开插件

点击左侧边栏的 🎬 图标，或使用命令面板搜索"打开我的影视"

### 2. 在日记中记录

在 `journals/2024-01-11.md` 文件中添加：

```markdown
# 2024年1月11日

今天看了几部好片：
- #video/movie 《肖申克的救赎》 经典之作，值得反复观看
- #video/tv 《权力的游戏》 史诗级剧集
- #video/variety 《向往的生活》 轻松愉快
```

### 3. 查看记录

1. 回到插件界面
2. 点击"刷新数据"按钮
3. 选择年份和月份
4. 查看观看记录和统计

## 标签格式

默认格式：`#video/类型 标题 评论`

支持的类型：
- `movie` - 电影
- `tv` - 电视剧
- `variety` - 综艺

## 自定义配置

点击"配置"按钮可以：

### 基础设置
- 修改应用名称（如"观影记录"）
- 修改视频标签（如 `#watch`、`#看片`）

### 类型管理
- 添加新类型（如"动漫"、"纪录片"）
- 修改现有类型
- 删除不需要的类型

## 示例配置

### 使用中文标签

配置：
```json
{
    "appName": "观影记录",
    "videoTag": "#看片",
    "videoTypes": {
        "电影": "电影",
        "剧集": "电视剧",
        "综艺": "综艺"
    }
}
```

使用：
```markdown
- #看片/电影 《肖申克的救赎》 经典
- #看片/剧集 《权力的游戏》 史诗
```

### 添加更多类型

在配置中添加：
- `anime` - 动漫
- `documentary` - 纪录片
- `short` - 短片

使用：
```markdown
- #video/anime 《你的名字》 新海诚的代表作
- #video/documentary 《地球脉动》 震撼
- #video/short 《调音师》 反转精彩
```

## 常见问题

### Q: 看不到记录？

检查：
1. 日记文件是否在 `journals` 文件夹
2. 文件名是否是 `yyyy-mm-dd.md` 格式
3. 标签格式是否正确
4. 点击"刷新数据"按钮

### Q: 如何修改标签？

1. 点击"配置"按钮
2. 修改"视频标签"
3. 保存配置
4. 在日记中使用新标签

### Q: 支持移动端吗？

支持！插件在移动端也可以正常使用。
