## 配置示例

### 配置1：使用中文标签

```json
{
    "appName": "观影记录",
    "videoTag": "#看片",
    "videoTypes": {
        "电影": "电影",
        "剧集": "电视剧",
        "综艺": "综艺"
    }
}
```

使用：
```markdown
- #看片/电影 《肖申克的救赎》 经典
- #看片/剧集 《权力的游戏》 史诗
```

### 配置2：添加更多类型

```json
{
    "appName": "我的影视",
    "videoTag": "#video",
    "videoTypes": {
        "movie": "电影",
        "tv": "电视剧",
        "variety": "综艺",
        "anime": "动漫",
        "documentary": "纪录片",
        "short": "短片"
    }
}
```

使用：
```markdown
- #video/anime 《你的名字》 新海诚的代表作
- #video/documentary 《地球脉动》 震撼的自然纪录片
- #video/short 《调音师》 反转精彩
```

### 配置3：简化标签

```json
{
    "appName": "追剧日志",
    "videoTag": "#v",
    "videoTypes": {
        "m": "电影",
        "t": "剧集",
        "v": "综艺"
    }
}
```

使用：
```markdown
- #v/m 《盗梦空间》 烧脑
- #v/t 《绝命毒师》 精彩
- #v/v 《向往的生活》 治愈
```

## 实际使用场景

### 场景1：电影爱好者

每次看完电影后在日记中记录：
```markdown
# 2024-01-11

今天看了诺兰的《盗梦空间》
- #video/movie 《盗梦空间》 ⭐⭐⭐⭐⭐
  多层梦境的设计太精妙了，每一层都有不同的时间流速。
  莱昂纳多的演技也很棒，最后的陀螺到底停没停？
  配乐：Hans Zimmer 的配乐太震撼了！
```

### 场景2：追剧记录

记录追剧进度和感受：
```markdown

  开局就很精彩，人物众多但不混乱。

# 2024-01-12

- #video/tv 《权力的游戏》S01E04-E07
  剧情渐入佳境，龙妈的成长线很精彩。
  小恶魔的智慧令人佩服。
```

记录看过的综艺：

晚饭时间看综艺放松：

  慢节奏的生活很治愈，适合放松。

## 技巧和建议

### 1. 保持记录习惯
- 看完后立即记录，趁记忆新鲜
- 记录真实感受，不要敷衍

### 2. 详细但不冗长
- 记录关键点和感受
- 不需要写影评，简短即可

### 3. 使用标签分类
- 可以添加额外标签：`#推荐` `#重温` `#经典`
- 方便后续搜索和回顾

### 4. 定期回顾
- 每月查看观看统计
- 了解自己的观影习惯
- 发现喜欢的类型和导演

### 5. 分享推荐
- 记录值得推荐的作品
- 可以导出分享给朋友
